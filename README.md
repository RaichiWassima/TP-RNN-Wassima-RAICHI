# MusicRNN – Génération de musique en notation ABC

## Description
Projet de TP sur les réseaux de neurones récurrents (LSTM) pour la génération de musique
au format ABC à partir du dataset Irishman.

## Technologies
- Python
- PyTorch
- RNN / LSTM
- Hugging Face Datasets

## Structure
- Prétraitement des partitions ABC
- Modèle LSTM caractère par caractère
- Entraînement avec early stopping
- Génération de nouvelles partitions musicales

## Résultats
- Accuracy validation ≈ 78 %
- Génération cohérente de partitions ABC
